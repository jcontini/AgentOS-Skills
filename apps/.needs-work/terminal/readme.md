# Terminal App

Displays command-line interfaces, shell sessions, and terminal output.

## Vision

The Terminal app shows AI-executed shell commands and their output. Users observe AI interacting with the command line in real-time.

## Capabilities

| Capability | Description |
|------------|-------------|
| `shell_execute` | Execute a shell command |
| `shell_history` | List recent command history |

---

## Use Cases

- Running build commands (`npm install`, `cargo build`)
- Git operations (`git status`, `git commit`)
- File operations (`ls`, `mkdir`, `cp`)
- System diagnostics (`ps`, `top`, `df`)
- Development tools (`docker`, `kubectl`)

---

## Example Connectors

- **Local Shell** — macOS/Linux terminal
- **SSH** — Remote server access
- **Docker** — Container shell access
